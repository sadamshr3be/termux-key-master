## [+] Installation :

* ```apt update```
* ```apt install git -y```
* ```git clone https://github.com/sadamshr3be/termux-key-master```
* ```cd termux-key```
* ```bash key.sh```

### Or ; يمكنك تثبيتها بهذا الامر
```
apt update && apt install git -y && git clone https://github.com/sadamshr3be/termux-key-master && cd termux-key && bash key.sh
```
